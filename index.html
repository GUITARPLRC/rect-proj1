Directions

  • To add rectangle: Click add rectangle
  • To clear the area: Click clear board
  • To change color or remove rectangle:
    • First, select either change color or remove rectangle
    • Second, double click to perform action
  • To save layout:
    • First, enter a name for the layout
    • Second, click save layout
  • To load a saved layout: Select from dropdown list
  • To delete a saved layout:
    • First, select saved layout from dropdown
    • Second, click delete selected layout
